# 智慧创造者网站

这是一个AI文案助手平台的演示网站。

## 🎯 项目简介
智慧创造者是一个专业的AI文案助手平台，帮助用户快速生成高质量的营销文案、社交媒体内容、产品描述等。

## 📋 功能特点
- **AI文案生成**：多种文案模板和风格
- **多套餐选择**：基础/专业/企业三个套餐
- **扫码支付集成**：支持支付宝/微信支付
- **响应式设计**：完美适配手机和电脑
- **专业客服支持**：人工审核和即时开通

## 🚀 快速开始

### 1. 本地测试
```bash
# 直接在浏览器中打开
双击 index.html
```

### 2. 部署到互联网
#### 使用GitHub Pages（推荐）
1. 创建GitHub账号（如果还没有）
2. 新建仓库：zhihuichuangzhe-website
3. 上传所有文件到仓库
4. 启用GitHub Pages功能
5. 访问：https://你的用户名.github.io/zhihuichuangzhe-website

#### 使用自定义域名
1. 购买域名：zhihuichuangzhe.com
2. 在GitHub Pages设置中添加域名
3. 配置DNS解析
4. 访问：https://zhihuichuangzhe.com

## 🔧 配置说明

### 支付功能配置
要启用真实支付功能，需要：

1. **准备收款码图片**
   - 支付宝收款码：alipay-qrcode.jpg
   - 微信收款码：wechat-qrcode.jpg
   - 建议尺寸：300x300像素

2. **修改支付页面**
   - 复制 payment-demo.html 为 payment.html
   - 替换二维码图片路径
   - 更新客服联系方式
   - 测试支付流程

3. **客服配置**
   - 微信客服：your-wechat-id
   - 邮箱客服：support@your-domain.com
   - 工作时间：09:00-21:00

### 网站信息配置
修改 `index.html` 中的以下内容：
- 网站标题和描述
- 联系方式
- 公司信息
- 服务条款链接
